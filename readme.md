# Neodock Recipes
A PHP repository for cooking recipes

### Overview
This project was developed using Neodock's opensource PHP framework.  It supports SQL Server as a backend database.  It requires at least PHP 8.3 or newer.

### System Requirements
* PHP 8.3 or newer running on IIS and Windows Server 2019 or newer
* SQL Server 2019, Standard or Enterprise, or newer
